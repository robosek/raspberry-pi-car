# raspberry_pi_car
Raspberry pi car main control script

![Racing car](car.gif)

## Assembly
### Motors raspberry connection
- GPIO 27 - IN 1
- GPIO 22 - IN 2
- GPIO 26 - IN 3
- GPIO 17 - IN 4

### Motors controller connection
#### Left motor:
- left side + 
- right side -

#### Right motor:
- left side - 
- right side +
